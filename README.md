# Duo Lite Addon (Demo)

Addon de exemplo para Stremio, feito para ser simples e fácil de editar.

## Como funciona

- Recurso: `stream`
- Tipos: `movie`, `series`
- Não tem catálogos (por enquanto)
- Responde **apenas** quando o Stremio pede o filme com ID `tt1254207`

Esse ID é do filme de teste **Big Buck Bunny**.

## URL de instalação

Depois de subir no Render, use esta URL no Stremio:

```text
https://SEU-NOME.onrender.com/manifest.json
